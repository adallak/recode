# recode
## Introduction

The `recode` is a complementary package to the book titled `Regressions in Covariances, Dependencies and Graphs`.

## Installation

To install the latest version from Github, use

```s
library(devtools)
devtools::install_github("adallak/recode")
```

## Usage
The package contains packages and functions used in Hands-on sections in the book. 
```s
library(recode)
```
